🎯 Ключові можливості

🎯

Виявлення підробної місткості

  • Перевірка 100 файлами (по 1% місткості кожен)
  • Випадкові позиції перевірки при кожному скануванні
  • Виявляє витончені підроблені контролери
  • Швидкий сирий зонд через прямий LBA-доступ (~1 хв.)
⚡

Тестування продуктивності

  • Реальне вимірювання швидкості читання/запису
  • Оптимізований стримінг для великих файлів
  • Відстеження прогресу з розрахунком ETA
  • Налаштовувані розміри файлів (1 МБ – 10 ГБ)
🔍

Управління дублікатами

  • Ідентифікація за хешем MD5
  • Кілька режимів вибору (найстаріші/найновіші/abc)
  • Гнучкі дії (видалити / перемістити)
  • Кешування хешів для пришвидшення повторних сканувань
📋

Операції копіювання

  • Відстеження прогресу з ETA
  • Захист від таймаутів для пошкоджених файлових систем
  • Збереження метаданих (права та часові мітки)
  • Підтримка пристроїв, тек та мережевих ресурсів
🧹

Швидке очищення

  • Ультра-швидко: видалити і перестворити теку (мс)
  • Стандартний резервний метод із відображенням прогресу
  • Збереження вихідних прав та часових міток
  • Працює з системними теками та обмеженим доступом
🛡️

Функції безпеки

  • Високошвидкісне безпечне видалення (4,7+ ГБ/с)
  • Операції заповнення з паралельним записом
  • Пакетна обробка кількох цілей
  • Повна історія операцій (JSON-лог)

⚡ Швидкий старт

Найчастіші завдання

# Перевірити підробку USB/SD карти
filedo E: test del

# Тест продуктивності диска
filedo C: speed 100

# Безпечне очищення вільного місця
filedo D: fill 1000 del

# Пошук та управління дублікатами
filedo C: check-duplicates
filedo D: cd old del

Копіювання & Очищення

# Копіювання з відстеженням прогресу
filedo folder C:\Source copy D:\Backup
filedo device E: copy F:\Archive

# Швидке очищення вмісту теки
filedo folder C:\Temp wipe

# Інформація про пристрій
filedo C: info
filedo D: short
