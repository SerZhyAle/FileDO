🎯 Fonctionnalités Principales

🎯

Détection de Fausse Capacité

  • Vérification avec 100 fichiers (1% de capacité chacun)
  • Vérification à positions aléatoires à chaque scan
  • Détecte les faux contrôleurs sophistiqués
  • Sonde rapide via accès direct LBA (~1 min, Admin requis)
⚡

Test de Performance

  • Mesure réelle de vitesse lecture/écriture
  • Streaming optimisé pour les grands fichiers
  • Suivi de progression avec calcul ETA
  • Tailles de fichiers configurables (1 Mo – 10 Go)
🔍

Gestion des Doublons

  • Identification par hachage MD5
  • Plusieurs modes de sélection (plus ancien/plus récent/abc)
  • Actions flexibles (supprimer / déplacer)
  • Mise en cache des hachages pour les scans répétés
📋

Opérations de Copie

  • Suivi de progression avec ETA
  • Protection contre les délais d'attente (systèmes endommagés)
  • Préservation des métadonnées (permissions & horodatages)
  • Compatible avec appareils, dossiers et partages réseau
🧹

Effacement Rapide

  • Ultra-rapide : supprimer & recréer le dossier (ms)
  • Méthode de secours avec affichage de progression
  • Préservation des permissions & horodatages originaux
  • Fonctionne avec les dossiers système & accès restreints
🛡️

Fonctionnalités de Sécurité

  • Effacement sécurisé haute vitesse (4,7+ Go/s)
  • Opérations de remplissage avec écriture parallèle
  • Traitement par lot pour plusieurs cibles
  • Historique complet des opérations (journal JSON)

⚡ Démarrage Rapide

Tâches Les Plus Courantes

# Vérifier la contrefaçon d'une clé USB/carte SD
filedo E: test del

# Test de performance du disque
filedo C: speed 100

# Nettoyage sécurisé de l'espace libre
filedo D: fill 1000 del

# Recherche et gestion des doublons
filedo C: check-duplicates
filedo D: cd old del

Copie & Nettoyage

# Copie avec suivi de progression
filedo folder C:\Source copy D:\Backup
filedo device E: copy F:\Archive

# Nettoyage rapide des dossiers
filedo folder C:\Temp wipe

# Afficher les informations du disque
filedo C: info
filedo D: short

📥 Installation

  1. Téléchargez filedo.exe depuis la page des Releases
  2. Optionnel : filedo_win.exe pour l'interface graphique VB.NET
  3. Exécutez depuis la ligne de commande : filedo <cible> <opération>
