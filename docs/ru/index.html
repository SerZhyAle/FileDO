🎯 Ключевые возможности

🎯

Обнаружение поддельной ёмкости

  • Проверка 100 файлами (по 1% ёмкости каждый)
  • Случайные позиции проверки при каждом сканировании
  • Выявляет изощрённые поддельные контроллеры
  • Быстрый сырой зонд через прямой LBA-доступ (~1 мин.)
⚡

Тестирование производительности

  • Реальное измерение скорости чтения/записи
  • Оптимизированный стриминг для больших файлов
  • Отслеживание прогресса с расчётом ETA
  • Настраиваемые размеры файлов (1 МБ – 10 ГБ)
🔍

Управление дубликатами

  • Идентификация по хешу MD5
  • Несколько режимов выбора (старейшие/новейшие/abc)
  • Гибкие действия (удалить / переместить)
  • Кэширование хешей для ускорения повторных сканирований
📋

Операции копирования

  • Отслеживание прогресса с ETA
  • Защита от таймаутов для повреждённых файловых систем
  • Сохранение метаданных (права и временны́е метки)
  • Поддержка устройств, папок и сетевых ресурсов
🧹

Быстрая очистка

  • Ультра-быстро: удалить и пересоздать папку (мс)
  • Стандартный резервный метод с отображением прогресса
  • Сохранение исходных прав и временны́х меток
  • Работает с системными папками и ограниченным доступом
🛡️

Функции безопасности

  • Высокоскоростное безопасное удаление (4,7+ ГБ/с)
  • Операции заполнения с параллельной записью
  • Пакетная обработка нескольких целей
  • Полная история операций (JSON-лог)

⚡ Быстрый старт

Наиболее частые задачи

# Проверить поддельность USB/SD карты
filedo E: test del

# Тест производительности диска
filedo C: speed 100

# Безопасная очистка свободного места
filedo D: fill 1000 del

# Поиск и управление дубликатами
filedo C: check-duplicates
filedo D: cd old del

Копирование & Очистка

# Копирование с отслеживанием прогресса
filedo folder C:\Source copy D:\Backup
filedo device E: copy F:\Archive

# Быстрая очистка содержимого папки
filedo folder C:\Temp wipe

# Информация об устройстве
filedo C: info
filedo D: short

📥 Установка

  1. Скачайте filedo.exe со страницы Releases
  2. Опционально: filedo_win.exe для графического интерфейса VB.NET
  3. Запускайте из командной строки: filedo <цель> <операция>
