🎯 Key Features

🎯

Fake Capacity Detection

  • 100-file verification (1% capacity each)
  • Random position verification every scan
  • Defeats sophisticated fake controllers
  • Fast raw probe via direct LBA access (~1 min)
⚡

Performance Testing

  • Real-world read/write speed measurement
  • Memory-optimized streaming for large files
  • Progress tracking with ETA calculations
  • Configurable file sizes (1 MB – 10 GB)
🔍

Duplicate Management

  • MD5 hash-based identification
  • Multiple selection modes (oldest/newest/abc)
  • Flexible actions (delete / move)
  • Hash caching for faster repeated scans
📋

Copy Operations

  • Progress tracking with ETA
  • Timeout protection for damaged filesystems
  • Preserves metadata (permissions & timestamps)
  • Works with devices, folders, network shares
🧹

Fast Wipe

  • Ultra-fast: delete & recreate folder (ms)
  • Standard fallback with progress display
  • Preserves original permissions & timestamps
  • Works with system folders & restricted access
🛡️

Security Features

  • High-speed secure wiping (4.7+ GB/s)
  • Fill operations with parallel writing
  • Batch processing for multiple targets
  • Full operation history (JSON log)

⚡ Quick Start

Most Common Tasks

# Check if USB/SD card is fake
filedo E: test del

# Test drive performance
filedo C: speed 100

# Secure wipe free space
filedo D: fill 1000 del

# Find & manage duplicates
filedo C: check-duplicates
filedo D: cd old del

Copy & Wipe

# Copy with progress tracking
filedo folder C:\Source copy D:\Backup
filedo device E: copy F:\Archive

# Fast wipe folder contents
filedo folder C:\Temp wipe

# Show device info
filedo C: info
filedo D: short

📥 Installation

  1. Download filedo.exe from the Releases page
  2. Optionally download filedo_win.exe for the VB.NET GUI
  3. Run from the command line: filedo <target> <operation>

💻 GUI Application

FileDO GUI (filedo_win.exe) is a VB.NET Windows Forms application providing a user-friendly interface with visual target selection, operation dropdowns, real-time output, and browse dialogs — all backed by the same CLI engine.

📄 Compare & Cleanup

# Compare two folders and save report
filedo compare D:\Data E:\Backup

# Compare and delete (permanent, no confirmation)
filedo cmp D:\Data E:\Backup del source   # delete in Source if exists in Target
filedo cmp D:\Data E:\Backup del old      # delete older side (by mtime)
filedo cmp D:\Data E:\Backup del small    # delete smaller side
